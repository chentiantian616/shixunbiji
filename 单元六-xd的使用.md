# UI和UX(UE)

**ui**：user interface 用户界面：以图形，文字，排版，配色为中心

**ux**：userexperieence 用户体验：以用户为中心。偏向的心理层面。**调研，头脑风暴，ui执行阶段**



# XD是什么

 **xd是一款设计功能强大的平台，易用平台。可以进行以下设计：**

1:网站设计：WUI

2：app设计：appui

3：语音设计：vui

4：游戏设计：game ui

**利用xd进行界面图形化设计以外，还可以创建原型和共享。**

XD是一个原型设计工具。

1：图形化界面工具

2：交互设计

# 原型的设计工具

**市场上原型设计工具特别多，可以分为两类：**

以图形设计为主：XD， sketch，磨刀（国产）等

设计原型交互逻辑为主：Axure,蓝湖等
